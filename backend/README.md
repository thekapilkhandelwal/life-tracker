# Life Tracker

Money, career, and travel tracker. Set Space secrets:

- `MONGODB_URI`
- `JWT_SECRET`
- `FRONTEND_URL` (this Space URL)
- `COOKIE_SECURE=true`
- `COOKIE_SAME_SITE=None`
- `DEMO_LOGIN_ENABLED=true`
- `GOOGLE_OAUTH_ENABLED=true` (after adding Google client id/secret)
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`
